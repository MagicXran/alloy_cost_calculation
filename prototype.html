核心公式与解释默认折叠
以下公式用于解释页面结果，不替代现场工艺规程；所有计算均基于当前输入的炉次重量、残余成分、目标范围、合金成分、收得率、价格和袋重。

通用元素增量

ΔE_i = x_i × alloy_E_i × recovery_E_i / 1000
x_i 为第 i 种合金加入量 kg/t；合金成分按百分数录入，除以 1000 后换算为钢水中的百分含量增量。

通用成分约束

target_min_E ≤ residual_E + ΣΔE_i ≤ target_max_E
最终成分等于残余成分加全部合金贡献；P、S 这类只有上限的元素只校核最大值。

控元素约束

residual_E + ΣΔE_i ≤ control_E - control_margin
控 Si 或控 C 启用后，该元素不再使用 min/max 范围，只保留单值上限约束。

通用吨钢成本

cost = Σ(price_i / 1000 × x_i)
价格单位为元/吨，加入量单位为 kg/t，因此除以 1000 后得到每吨钢合金成本。

经验方案模型

x_rule = conservative_milp(low_carbon_alloys) 或 sequence_greedy
经验方案优先使用低碳合金集合做保守整袋求解；若保守集合不可行，则按内部路线逐步补足 Cr、Si、Mn、C，并由成分校核标记是否合格。

LP 理论下限模型

min cᵀx, s.t. A x ≤ b, 0 ≤ x_i ≤ max_i
LP 把所有合金加入量都视为连续变量，求成本理论下限；它用于看极限成本，不等于现场可直接执行的整袋投料单。

MILP 整袋模型

x_i = bags_i × bag_size_i / heat_weight, bags_i ∈ Z
对启用整袋投料的合金增加整数袋约束，连续投料合金仍保持连续变量；分支定界在成分合格前提下搜索最低成本现场方案。
